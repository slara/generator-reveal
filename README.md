# Generator-reveal

A Reveal.js generator for Yeoman.

## Getting started
- Make sure you have [yo](https://github.com/yeoman/yo) installed:
    `git clone https://github.com/slara/generator-reveal`
    `cd generator-reveal`
- link the generator: `npm link`
- Go to your new project directory
- Run: `yo reveal`
